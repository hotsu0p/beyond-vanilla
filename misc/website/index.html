How to Install

  1. Download the beyond-vanilla.zip file
  2. Open your Minecraft shaders folder
  3. Unzip the beyond-vanilla.zip contents into the shaders folder
  4. Fire up Minecraft and head to Options > Video Settings
  5. From the Shaders dropdown, pick Beyond Vanilla
  6. Click Done and get ready to be blown away!
